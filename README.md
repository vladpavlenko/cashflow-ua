# CashFlow UA — Інструкція розгортання

## Файли

| Файл | Опис |
|------|------|
| `index.html` | Головний файл гри |
| `manifest.json` | PWA маніфест (іконка, назва) |

---

## Безкоштовне розгортання на GitHub Pages (10 хвилин)

1. Зареєструйся на [github.com](https://github.com)
2. Натисни **"New repository"** → назви `cashflow-ua` → **Public** → Create
3. Завантаж всі 4 файли (Upload files)
4. Перейди: **Settings → Pages → Source → main → Save**
5. Через 1-2 хвилини гра буде доступна за адресою:
   `https://ТВІЙ_НІКНЕЙМ.github.io/cashflow-ua/`
6. **Поділись посиланням з друзями** — вони також зможуть встановити!

---

## Встановлення на iPhone (кожен гравець)

1. Відкрий посилання у **Safari** (не Chrome!)
2. Натисни кнопку **Поділитись** (квадрат зі стрілкою вгору)
3. Прокрути вниз → **"На екран «Початок»"**
4. Назви `CashFlow` → **Додати**

Готово! Застосунок з'явиться на головному екрані як звичайний додаток.

---

## Іконка (опційно)

Для гарної іконки на iOS:
1. Відкрий [cloudconvert.com/svg-to-png](https://cloudconvert.com/svg-to-png)
2. Завантаж `icon.svg`, розмір 192×192 → збережи як `icon-192.png`
3. Зроби ще раз: розмір 512×512 → збережи як `icon-512.png`
4. Додай обидва файли в репозиторій поруч з іншими

## Що вміє поточна версія (Раунд 1)

- ✅ Вибір освіти (університет / технікум / без освіти)
- ✅ Вибір спеціальності
- ✅ Автоматична стипендія $30 для студентів
- ✅ Таблиця доходів з формулою: (Базова + Дод.) × Множник
- ✅ Таблиця витрат: житло (з варіантами оплати), їжа, зв'язок, транспорт
- ✅ Сім'я: дружина + до 3 дітей
- ✅ Кредити: Банк (3%) та Мікрокредит (30%)
- ✅ Таблиця курсів (збільшують Додаткову оплату)
- ✅ Таблиця тренінгів (збільшують Множник)
- ✅ Завершення раунду: готівка оновлюється автоматично
- ✅ Зберігає стан між сесіями (localStorage)
- ✅ Офлайн-режим

## Наступні кроки (Раунд 2+)

Коли будеш готовий, напиши мені:
- Механіку влаштування на роботу
- Механіку старту бізнесу / стартапу
- Події / картки випадкових обставин
- Будь-які зміни до вже реалізованого

Просто відправ нову розмову зі словами: "Продовжуємо CashFlow, ось що треба додати..."
