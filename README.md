## Крок 1 — Firebase (безкоштовно, ~10 хвилин)

### 1.1 Створи проєкт
1. Іди на [console.firebase.google.com](https://console.firebase.google.com)
2. **Додати проєкт** → назви `cashflow-ua` → Continue → Continue
3. Google Analytics → вимкни → Створити проєкт

### 1.2 Realtime Database
1. В лівому меню: **Build → Realtime Database**
2. **Create Database** → Вибери регіон `europe-west1` → **Next**
3. Режим → **Start in test mode** → **Enable**

### 1.3 Отримай конфіг
1. Шестерня ⚙️ → **Project settings**
2. Прокрути вниз до **Your apps**
3. Натисни `</>` (Web) → Назви `cashflow-pwa` → Register app
4. Скопіюй блок `firebaseConfig`:

```javascript

  apiKey: "AIzaSyAOjnui8wVleQ-pNbGculhezKHrnEgLQcw",
  authDomain: "cashflow-ua.firebaseapp.com",
  databaseURL: "https://cashflow-ua-default-rtdb.europe-west1.firebasedatabase.app",
  projectId: "cashflow-ua",
  storageBucket: "cashflow-ua.firebasestorage.app",
  messagingSenderId: "883578570835",
  appId: "1:883578570835:web:dc441ce423516663575ea6"
```

### 1.4 Встав конфіг у файли
Відкрий **`index.html`** і **`gm.html`** у текстовому редакторі.
Знайди блок `const firebaseConfig = {` і замінити всі `"ТВІЙ_..."` рядки своїми значеннями.

> ⚠️ Зроби це в **обох** файлах — конфіг однаковий.

## Крок 2 — Розгортання (GitHub Pages)

1. Зареєструйся на [github.com](https://github.com)
2. **New repository** → `cashflow-ua` → Public → Create
3. Завантаж усі файли: `index.html`, `gm.html`, `manifest.json`, `sw.js`
4. **Settings → Pages → Source → main → Save**
5. Через 2 хв посилання буде:
   - Гравці: `https://НІКНЕЙМ.github.io/cashflow-ua/`
   - Ведучий: `https://НІКНЕЙМ.github.io/cashflow-ua/gm.html`

## Крок 3 — Встановлення на телефон

### iPhone (Safari)
1. Відкрий посилання у **Safari**
2. Кнопка «Поділитись» → **На екран «Початок»**
3. Назви → **Додати**

### Android (Chrome)
1. Відкрий посилання у **Chrome**
2. Chrome автоматично запропонує **«Встановити додаток»**
3. Або: ⋮ меню → **Додати на головний екран**

## Як грати

### Ведучий (GM)
1. Відкрий `gm.html`
2. Натисни **«Створити нову»** — отримаєш 4-літерний код
3. Поділись кодом з гравцями
4. Бачиш усіх гравців в реальному часі, їх готівку, доходи, витрати
5. Червоні значки ⚠️ з'являються при аномаліях (від'ємна готівка, перевищення лімітів)
6. Натисни **«Всі деталі»** на картці гравця — розкриється повний список доходів, витрат, курсів, тренінгів

### Гравець
1. Відкрий `index.html`
2. Введи **код кімнати** від ведучого
3. Введи ім'я, обери освіту та спеціальність
4. Грай — кожна зміна автоматично синхронізується з ведучим
